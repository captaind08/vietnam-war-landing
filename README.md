# Vietnam War Evidence Story Landing Page

A Vietnamese-first bilingual educational landing page about the Vietnam War, built as a story with a beginning, middle, and end.

## What it includes

- Vietnamese-first experience with English toggle
- Clear 5W section: who, what, where, when, why
- Story structure:
  - Beginning: independence, colonialism, Geneva, partition
  - Middle: South Vietnam legitimacy crisis, Gulf of Tonkin, U.S. escalation, Tet, Laos/Cambodia
  - End: Paris Peace Accords, Fall of Saigon, reunification, re-education, refugees, Agent Orange, UXO, diaspora memory
- Archival/public-domain image blocks with visible captions and source links
- Illustrated map section
- Contextual timeline
- Source library for verification

## Tone and purpose

This page is designed as educational material, not propaganda. It aims to respect multiple Vietnamese memories and help readers understand the evidence before reaching conclusions.

## Sources included

The page links to reputable sources such as:

- U.S. National Archives
- Foreign Relations of the United States / Office of the Historian
- NSA Gulf of Tonkin declassified materials
- CIA Reading Room
- Wilson Center Digital Archive
- Naval History and Heritage Command
- U.S. Department of Veterans Affairs Agent Orange resources
- Texas Tech Virtual Vietnam Archive
- UNHCR refugee materials
- Britannica

## Publishing with GitHub Pages

Because this is a single static `index.html` file, it can be published directly with GitHub Pages:

1. Go to the repository Settings.
2. Open Pages.
3. Under Build and deployment, choose `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save.

GitHub will generate a public URL after deployment.
